# ATPSome
##关于
尝试用Node.js来写了一个微信公众号   
可以查询一些网球（ATP）相关的咨询

##部署
先把config.demo.js重命名为config.js,然后在根目录
```
npm install 
```
then
```
npm run build
```
then
```
node server
```
